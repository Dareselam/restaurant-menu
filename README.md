# Il Ristorantino Italiano – Streamlit Menu App

An interactive restaurant menu built with **Streamlit** and custom HTML/CSS styling.  
This app showcases authentic Italian cuisine with expandable sections for antipasti, pasta, risotti, seafood, drinks, spirits, and craft beers.

# Features
- Custom styled header** with restaurant branding
- Expandable sections** for each menu category
- Clear display of **prices and descriptions** for every dish
-  Highlighted section for **Cane Nero Brewery** with contact details
-  Informational notes (e.g., fish dishes served with salad)
- Responsive layout using Streamlit’s wide mode


 # Project Structure
- 'app.py' → Main Streamlit application code
- 'requirements.txt'→ Python dependencies (e.g., `streamlit`)
